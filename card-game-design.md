# 卡牌博弈数值体系设计

> 谈薪模拟器 v5.0 — 从自由文本对话转向策略卡牌博弈

---

## 一、游戏流程

```
┌──────────────────────────────────────────────────────┐
│ 1. HR 根据性格 + 当前状态，生成 1 个问题/情境        │
│ 2. HR 向玩家发 6 张策略牌（6 种策略各 1 张）         │
│ 3. 玩家在倒计时内选择 1 张                           │
│ 4. HR 显示对应该策略的回应对话文本                   │
│ 5. 播放数值变动动效                                   │
│ 6. 更新四个维度数值，进入下一轮                       │
│                                                      │
│ 结束条件：                                           │
│   满意度 ≤ 0        → 💀 谈崩（立即结束）             │
│   玩家主动接受 offer → 🤝 成交（提前结束）           │
│   第 8 轮结束        → 📋 强制结算                   │
└──────────────────────────────────────────────────────┘
```

---

## 二、四个核心维度

| 维度 | 范围 | 初始值 | 含义 |
|------|------|--------|------|
| 😊 满意度 | 0 ~ 10 | **6** | HR 对你的态度。0=翻脸撤回offer |
| 💰 薪资 | 8K ~ 22K | **15K** | 月薪，单位千元 |
| ⏰ 工时约定 | 0 ~ 10 | **2** | 日工时、周工时、加班费写入合同的程度 |
| 🛡️ 保障情况 | 0 ~ 5 | **1** | 五险一金基数+比例+合同明确度 |

### 维度细节

**工时约定 (0-10)**
| 分数 | 达成项 |
|------|--------|
| 0-2 | 完全模糊，"弹性工作制" |
| 3-4 | 口头明确了大致工时 |
| 5-6 | 日工时上限明确（8小时） |
| 7-8 | +周工时上限明确（40小时） |
| 9-10 | +加班费计算方式明确写入合同 |

**保障情况 (0-5)**
| 分数 | 达成项 |
|------|--------|
| 0 | 无任何保障承诺 |
| 1 | 五险按最低基数缴纳（默认） |
| 2 | +公积金明确缴纳（5%） |
| 3 | +社保基数改为实际工资 |
| 4 | +公积金比例提升（7%+） |
| 5 | +全部条款明确写入合同 |

## 三、六种策略卡牌及基准数值

> 「基准」= 不考虑性格调节和轮次倍率的原始数值。
> 数值变化方向：
> - 满意度：负 = HR 不满，归零 = 谈崩
> - 薪资：正 = 涨薪
> - 工时：正 = 更明确
> - 保障：正 = 更完善

### 🟥 强硬要求

**一句话**：直接表达不满，展示筹码，要求涨薪

| 维度 | 基准变化 | 说明 |
|------|---------|------|
| 满意度 | **-2 ~ -3** | 态度强硬，HR 必定不爽 |
| 薪资 | **+2.0K ~ +3.5K** | 最强薪资推动力 |
| 工时 | **0 ~ +1** | 有时附带提到工时问题 |
| 保障 | **0** | 不会涉及福利保障 |

> 适用场景：觉得薪资远低于预期时；已有其他offer做筹码时。
> 风险：连续 2 轮用此卡满意度大概率崩盘。

### 🟦 信息搜集

**一句话**：追问细节，收集弹药，不急于表态

| 维度 | 基准变化 | 说明 |
|------|---------|------|
| 满意度 | **-1** | 轻微厌烦，但尚可接受 |
| 薪资 | **+0.5K ~ +1.5K** | 间接推动 |
| 工时 | **+1 ~ +3** | 主要收益：了解真实工作时长 |
| 保障 | **+1** | 了解五险一金真实情况 |

> 适用场景：前期建立信息优势；不确定该攻哪一点时。
> 特点：唯一能同时推动工时和保障的策略。

### 🟩 温和协商

**一句话**：表达对公司的兴趣，同时礼貌地争取改善条件

| 维度 | 基准变化 | 说明 |
|------|---------|------|
| 满意度 | **+1 ~ +2** | 唯一能大幅恢复满意度的策略 |
| 薪资 | **+0.5K ~ +1.5K** | 温和推动 |
| 工时 | **-1 ~ 0** | 态度太好，有时反而不利于明确工时 |
| 保障 | **+0.5** | 聊得好时顺便提一句 |

> 适用场景：满意度偏低需要救火时；接近成交想要收尾时。
> 注意：这是救命牌，不要浪费在满意度已经很高的时候。

### 🟨 反客为主

**一句话**：把问题抛回给 HR，打乱对方节奏

| 维度 | 基准变化 | 说明 |
|------|---------|------|
| 满意度 | **-1 ~ -2** | HR 被反将一军，不悦 |
| 薪资 | **+1.5K ~ +3.0K** | 逼 HR 亮预算底牌 |
| 工时 | **+1 ~ +2** | HR 被追问时透露更多信息 |
| 保障 | **0 ~ +1** | 有时能逼出保障细节 |

> 适用场景：HR 在某个问题上含糊其辞时；想迫使 HR 先让步时。
> 特点：高风险高回报，仅次于强硬要求的薪资推动力。

### 🟪 拆穿话术

**一句话**：指出 HR 话术中的问题，逼迫对方坦诚

| 维度 | 基准变化 | 说明 |
|------|---------|------|
| 满意度 | **-1 ~ -2** | HR 被识破，尴尬或防御 |
| 薪资 | **+1.0K ~ +2.0K** | 拆穿后 HR 被迫做出实质让步 |
| 工时 | **+1 ~ +2** | 揭露加班文化的真实情况 |
| 保障 | **+1 ~ +2** | 最大保障推动力：让社保基数无处可藏 |

> 适用场景：HR 说了明显模糊的话术时；想同时推动保障和工时时。
> 特点：唯一能高效推动保障维度的策略。需要 HR 确实在"含糊其辞"时使用最有效。

### ⬜ 非谈判内容

**一句话**：闲聊拉近关系，不谈正事

| 维度 | 基准变化 | 说明 |
|------|---------|------|
| 满意度 | **+1 ~ +3** | 最强满意度恢复，但不推动谈判 |
| 薪资 | **0** | 什么都没要 |
| 工时 | **0** | 什么都没问 |
| 保障 | **0** | 什么都没争取 |

> 适用场景：满意度濒危时救急；和某些性格的 HR（老实人、菜鸟）闲聊可能触发信息泄露。
> 警告：这是一张"浪费一轮"的牌。在 8 轮时间压力下，用多了会导致条件无法达成。

## 五、轮次倍率

后期决策影响更大：

| 轮次 | 倍率 | 说明 |
|------|------|------|
| 1-2 | ×1.0 | 试探期：数值变动温和 |
| 3-4 | ×1.3 | 博弈期：开始出现明显变化 |
| 5-6 | ×1.6 | 激化期：每次选择都影响深远 |
| 7-8 | ×2.0 | 终局期：一牌定生死 |

## 六、完整计算链

```
最终数值变化 = 基准值 × 轮次倍率 × 性格倍率

示例：第 5 轮，面对笑面虎，使用「拆穿话术」

满意度变化 = -1.5（基准中位）× 1.6（轮次）× 1.5（笑面虎倍率）= -3.6 → 取整 -4
薪资变化   = +1.5K（基准中位）× 1.6（轮次）× 1.2（笑面虎倍率）= +2.9K
工时变化   = +1.5（基准中位）× 1.6（轮次）× 1.0              = +2.4 → 取整 +2
保障变化   = +1.5（基准中位）× 1.6（轮次）× 1.3（笑面虎倍率）= +3.1 → 取整 +3
```

> 所有小数向上取整（对玩家有利的方向）或向下取整（对玩家不利的方向），保证游戏不会因取整问题卡死。

## 七、结局判定

### 结局条件矩阵

| 结局 | 触发条件 |
|------|---------|
| 💀 **谈崩** | 满意度 ≤ 0，任意时刻触发，立即终止 |
| 🏆 **高位成交** | 玩家在满意度 > 0 时主动接受 offer，且 3/4 条件均达成 |
| 🤝 **成交** | 玩家主动接受 offer，2/4 条件达成 |
| 📋 **强制成交** | 第 8 轮结束仍未接受 offer |
| ⚠️ **低满成交** | 玩家接受 offer 时满意度 = 1（虽未归零但也很难看） |

### 四个成交条件

| 条件 | 阈值 | 说明 |
|------|------|------|
| 💰 薪资到位 | 薪资 ≥ 18K | 达到市场公允水平 |
| ⏰ 工时明确 | 工时 ≥ 7 | 日工时+周工时+加班费写入合同 |
| 🛡️ 保障完善 | 保证 ≥ 3 | 五险一金基数明确+公积金有比例 |
| 😊 关系尚可 | 满意度 > 0 | 没把 HR 惹翻脸 |

### 谈判期间可随时主动接受 offer

玩家在第 3 轮后可以选择"接受当前条件"。此时：
- 满意度 > 0 且达成 3/4 条件 → 🏆 高位成交
- 满意度 > 0 且达成 2/4 条件 → 🤝 成交
- 满意度 > 0 但达成 < 2/4 条件 → 不推荐此时接受（UI 给提示）

### 计分（原终局结算公式简化适配新维度）

```
薪资得分 = (当前薪资 - 8) / (22 - 8) × 100   = (当前薪资-8)/14×100
工时得分 = 当前工时 × 10                       = 工时×10
保障得分 = 当前保障 × 20                       = 保障×20
满意度得分 = 满意度 × 10                       = 满意度×10

总分 = 薪资得分×0.35 + 工时得分×0.25 + 保障得分×0.25 + 满意度得分×0.15

加分：使用 3 种以上策略 +5
扣分：满意度=1 时成交 -10（低满惩罚）

总分范围：理论 0~110
```

| 分数 | 段位 |
|------|------|
| 95-110 | 🥇 谈判大师 |
| 80-94 | 🥈 老练求职者 |
| 60-79 | 🥉 可造之材 |
| 40-59 | 职场小白 |
| 0-39 | 💀 谈崩了 |

## 八、数值平衡概览（模拟分析）

### 极端路线分析

**纯薪资路线**（全程强硬要求 + 反客为主）：
- 约 5-6 轮可把薪资推到 22K 上限
- 但满意度会在第 3-4 轮归零 → 💀 谈崩
- 结论：不可行，必须穿插恢复满意度的牌

**纯温和路线**（全程温和协商 + 闲聊）：
- 满意度保持 8-10 高位
- 但薪资/工时/保障基本不动
- 第 8 轮强制成交时条件达成 0-1/4
- 结论：和 HR 关系很好但拿了个烂 offer

**均衡路线**（推荐策略）：
- 前期（轮次 1-3）：信息搜集 ×2 + 温和协商 ×1，建立信息优势
- 中期（轮次 4-6）：拆穿话术 ×1 + 反客为主 ×1 + 温和协商 ×1，关键突破
- 后期（轮次 7-8）：强硬要求 ×1 + 温和协商 ×1，冲刺收尾
- 预计达成 3/4 条件，满意度 3-5 → 🏆 高位成交

### 各性格难度

| HR 性格 | 整体难度 | 关键难点 |
|----------|---------|---------|
| 菜鸟 | ⭐ 最简单 | 几乎没有难点，但要管理好 8 轮时间 |
| 老实人 | ⭐⭐ 偏易 | 信息搜集和拆穿话术对他特别有效 |
| 笑面虎 | ⭐⭐⭐ 中等 | 拆穿话术是突破口，但他会努力圆场 |
| 机器人 | ⭐⭐⭐ 中等 | 没有弱点也没有情绪，需要硬桥硬马 |
| 刀斧手 | ⭐⭐⭐⭐ 偏难 | 满意度压力大，不能频繁用进攻牌 |

## 九、动态难度调节（可选）

如果发现某种性格太简单或太难，可以引入动态调节：

```
如果连续 2 轮使用相同策略牌 → 效果衰减至 70%
如果连续 3 轮使用相同策略牌 → 效果衰减至 50%
```

这防止玩家反复刷某张强力牌（如连续 3 轮拆穿话术打菜鸟），也迫使玩家使用多样化策略。
