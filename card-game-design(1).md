# 卡牌博弈数值体系设计

> 谈薪模拟器 v5.0 — 从自由文本对话转向策略卡牌博弈

---

## 一、游戏流程

```
┌──────────────────────────────────────────────────────┐
│ 1. HR 根据性格 + 当前状态，生成 1 个问题/情境        │
│ 2. HR 向玩家发 6 张策略牌（6 种策略各 1 张）         │
│ 3. 玩家在倒计时内选择 1 张                           │
│ 4. HR 显示对应该策略的回应对话文本                   │
│ 5. 播放数值变动动效                                   │
│ 6. 更新四个维度数值，进入下一轮                       │
│                                                      │
│ 结束条件：                                           │
│   满意度 ≤ 0        → 💀 谈崩（立即结束）             │
│   玩家主动接受 offer → 🤝 成交（提前结束）           │
│   第 8 轮结束        → 📋 强制结算                   │
└──────────────────────────────────────────────────────┘
```

## 二、四个核心维度

| 维度 | 范围 | 初始值 | 含义 |
|------|------|--------|------|
| 😊 满意度 | 0 ~ 10 | **6** | HR 对你的态度。0=翻脸撤回offer |
| 💰 薪资 | 8K ~ 22K | **15K** | 月薪，单位千元 |
| ⏰ 工时约定 | 0 ~ 10 | **2** | 日工时、周工时、加班费写入合同的程度 |
| 🛡️ 保障情况 | 0 ~ 5 | **1** | 五险一金基数+比例+合同明确度 |

### 维度细节

**工时约定 (0-10)**
| 分数 | 达成项 |
|------|--------|
| 0-2 | 完全模糊，"弹性工作制" |
| 3-4 | 口头明确了大致工时 |
| 5-6 | 日工时上限明确（8小时） |
| 7-8 | +周工时上限明确（40小时） |
| 9-10 | +加班费计算方式明确写入合同 |

**保障情况 (0-5)**
| 分数 | 达成项 |
|------|--------|
| 0 | 无任何保障承诺 |
| 1 | 五险按最低基数缴纳（默认） |
| 2 | +公积金明确缴纳（5%） |
| 3 | +社保基数改为实际工资 |
| 4 | +公积金比例提升（7%+） |
| 5 | +全部条款明确写入合同 |

## 三、六种策略卡牌及基准数值（v2.0 修正版）

> ⚠️ **平衡原则**：每张牌都是取舍。没有"全正面"的牌。想涨薪就得得罪 HR，想修复关系就得牺牲一轮进度。
>
> 「基准」= 不考虑性格调节和轮次倍率的原始数值。

---

### 策略总览（一图看懂）

| 策略 | 满意度 | 薪资 | 工时 | 保障 | 一句话定性 |
|------|--------|------|------|------|-----------|
| 🟥 强硬要求 | **-3** | **+2.5K** | 0 | 0 | 拿钱换命 |
| 🟦 信息搜集 | **-1** | **+0.5K** | **+1.5** | **+1** | 低调发育 |
| 🟩 温和协商 | **+1** | **+0.5K** | **-1** | 0 | 回血小亏 |
| 🟨 反客为主 | **-2** | **+2K** | **+1** | 0 | 剑走偏锋 |
| 🟪 拆穿话术 | **-2** | **+1K** | **+1** | **+1.5** | 直捣黄龙 |
| ⬜ 非谈判内容 | **+1.5** | 0 | 0 | 0 | 纯回血，零进度 |

---

### 🟥 强硬要求

**一句话**：直接表达不满，展示筹码，要求涨薪

| 维度 | 基准变化 | 说明 |
|------|---------|------|
| 满意度 | **-3** | 态度强硬，HR 必定不爽。这是最大单卡满意度惩罚 |
| 薪资 | **+2.5K** | 最强薪资推动力，但要付出关系代价 |
| 工时 | **0** | 不会涉及工时问题 |
| 保障 | **0** | 不会涉及福利保障 |

> **取舍分析**：每用一次消耗 3 点满意度，换 2.5K 薪资。初始 6 点满意度，纯强硬路线最多用 2 次就会归零（第 2 次用完满意度=0→💀）。如果中间穿插恢复，一张温和协商只回 1 点，回血速度远跟不上掉血速度。
>
> **适合**：满意度 ≥ 4 时才敢用的核武器。后期轮次倍率放大后尤其危险。

### 🟦 信息搜集

**一句话**：追问细节，收集弹药，不急于表态

| 维度 | 基准变化 | 说明 |
|------|---------|------|
| 满意度 | **-1** | 轻微厌烦，但尚可接受 |
| 薪资 | **+0.5K** | 间接推动，聊胜于无 |
| 工时 | **+1.5** | 主要收益：了解真实工作时长 |
| 保障 | **+1** | 了解五险一金真实情况 |

> **取舍分析**：最温和的推进牌。用 1 点满意度换工时+保障的稳定推进。不涨薪是硬伤，不能用来冲薪资条件。
>
> **适合**：前期必备。建立信息优势后才知道该攻哪。连续使用两次是安全的开局。

### 🟩 温和协商

**一句话**：表达对公司的兴趣，同时礼貌地争取改善条件

| 维度 | 基准变化 | 说明 |
|------|---------|------|
| 满意度 | **+1** | 恢复关系，但每轮只能回 1 点 |
| 薪资 | **+0.5K** | 聊胜于无的薪资推动 |
| 工时 | **-1** | ⚠️ 态度太好，HR 趁机收回工时承诺 |
| 保障 | **0** | 不会涉及保障话题 |

> **取舍分析**：仅回 1 点满意度——2 轮温和协商才回 2 点，而一轮强硬要求就扣 3 点。这意味回血速度永远追不上掉血速度。工时倒退 1 点更是惩罚——你太 nice 了，HR 觉得不需要在工时上让步。
>
> **适合**：满意度 ≤ 2 时的救命牌。不要用于冲薪资——+0.5K 太少了。

### 🟨 反客为主

**一句话**：把问题抛回给 HR，打乱对方节奏

| 维度 | 基准变化 | 说明 |
|------|---------|------|
| 满意度 | **-2** | HR 被反将一军，不悦 |
| 薪资 | **+2K** | 仅次于强硬要求的薪资推动力 |
| 工时 | **+1** | HR 被追问时透露工时信息 |
| 保障 | **0** | 对保障维度无帮助 |

> **取舍分析**：2 点满意度换 2K 薪资 + 1 工时。性价比高于强硬要求（少扣 1 点满意度），但薪资推动力也少了 0.5K。不能用来推动保障——如果你需要改善五险一金，这不是正确选择。
>
> **适合**：中等满意度（3-4）时的进攻选择。比强硬要求安全，比信息搜集激进。

### 🟪 拆穿话术

**一句话**：指出 HR 话术中的问题，逼迫对方坦诚

| 维度 | 基准变化 | 说明 |
|------|---------|------|
| 满意度 | **-2** | HR 被识破，尴尬或防御 |
| 薪资 | **+1K** | 拆穿后有所让步但薪资不是主要收益 |
| 工时 | **+1** | 揭露加班文化真实情况 |
| 保障 | **+1.5** | 最大保障推动力：让社保基数无处可藏 |

> **取舍分析**：唯一能同时推动工时和保障的牌，且保障推动力最强（+1.5）。但不涨薪（+1K 太少）——你需要靠其他牌来冲薪资条件。
>
> **适合**：保障和工时落后时的关键牌。搭配信息搜集使用可控制这两个维度。

### ⬜ 非谈判内容

**一句话**：闲聊拉近关系，不谈正事

| 维度 | 基准变化 | 说明 |
|------|---------|------|
| 满意度 | **+1.5** | 最大满意度恢复，但只比温和协商多 0.5 |
| 薪资 | **0** | ⚠️ 什么都没要——实打实地浪费了一轮 |
| 工时 | **0** | ⚠️ 什么都没问 |
| 保障 | **0** | ⚠️ 什么都没争取 |

> **取舍分析**：回 1.5 满意度但零进度。连续使用 2 轮 = 零进展 + 回 3 满意度 = 用 1/4 的总轮次换 3 点满意度。在 8 轮限时下，这是一个代价极高的选择。
>
> **适合**：满意度降到 1 时的绝望救援。非紧急情况必不用——用了意味着你必须在剩余轮次中用更激进的牌来追赶进度，而这又会扣满意度，形成恶性循环。
>
> ⚠️ **警告**：某些性格（机器人、刀斧手）闲聊效果打折，甚至无效。面对这些 HR 时闲聊是陷阱牌。

## 五、轮次倍率

> ⚠️ 轮次倍率是**双刃剑**——它放大收益也放大惩罚。后期一次错误的强硬要求可能直接让满意度归零。

| 轮次 | 倍率 | 说明 |
|------|------|------|
| 1-2 | ×1.0 | 试探期：犯错代价可控，适合信息搜集建立情报 |
| 3-4 | ×1.3 | 博弈期：满意度变化开始有感觉，一次进攻扣 2-4 点 |
| 5-6 | ×1.6 | 激化期：满意度惩罚加重——强硬要求一轮扣 4.8 满意度 |
| 7-8 | ×2.0 | 终局期：一牌定生死。强硬要求一轮扣 6 满意度 → 满意度 ≤ 2 则直接 💀 |

> **轮次倍率不是玩家的朋友。** 它让后期的进攻更强但也更危险——玩家必须在前期建立足够的满意度缓冲，才能在后期打出致命一击而不翻车。

## 六、完整计算链

```
最终数值变化 = 基准值 × 轮次倍率 × 性格倍率 × 事件倍率

示例：第 5 轮，面对笑面虎，使用「拆穿话术」

满意度变化 = -2.0（基准）× 1.6（轮次）× 1.5（笑面虎倍率）= -4.8 → 取整 -5
薪资变化   = +1.0K（基准）× 1.6（轮次）× 1.2（笑面虎倍率）= +1.92K
工时变化   = +1.0（基准）× 1.6（轮次）× 1.0              = +1.6 → 取整 +2
保障变化   = +1.5（基准）× 1.6（轮次）× 1.3（笑面虎倍率）= +3.12 → 取整 +3

⚠️ 第 5 轮用这张牌扣 5 满意度！如果当前满意度 ≤ 5，立即 💀。
```

> 取整规则：满意度向下取整（对玩家不利）、薪资/工时/保障向上取整（对玩家有利）。

## 七、结局判定

### 结局条件矩阵

| 结局 | 触发条件 |
|------|---------|
| 💀 **谈崩** | 满意度 ≤ 0，任意时刻触发，立即终止 |
| 🏆 **高位成交** | 玩家在满意度 > 0 时主动接受 offer，且 3/4 条件均达成 |
| 🤝 **成交** | 玩家主动接受 offer，2/4 条件达成 |
| 📋 **强制成交** | 第 8 轮结束仍未接受 offer |
| ⚠️ **低满成交** | 玩家接受 offer 时满意度 = 1（虽未归零但也很难看） |

### 四个成交条件

| 条件 | 阈值 | 说明 |
|------|------|------|
| 💰 薪资到位 | 薪资 ≥ 18K | 达到市场公允水平 |
| ⏰ 工时明确 | 工时 ≥ 7 | 日工时+周工时+加班费写入合同 |
| 🛡️ 保障完善 | 保证 ≥ 3 | 五险一金基数明确+公积金有比例 |
| 😊 关系尚可 | 满意度 > 0 | 没把 HR 惹翻脸 |

### 谈判期间可随时主动接受 offer

玩家在第 3 轮后可以选择"接受当前条件"。此时：
- 满意度 > 0 且达成 3/4 条件 → 🏆 高位成交
- 满意度 > 0 且达成 2/4 条件 → 🤝 成交
- 满意度 > 0 但达成 < 2/4 条件 → 不推荐此时接受（UI 给提示）

### 计分（原终局结算公式简化适配新维度）

```
薪资得分 = (当前薪资 - 8) / (22 - 8) × 100   = (当前薪资-8)/14×100
工时得分 = 当前工时 × 10                       = 工时×10
保障得分 = 当前保障 × 20                       = 保障×20
满意度得分 = 满意度 × 10                       = 满意度×10

总分 = 薪资得分×0.35 + 工时得分×0.25 + 保障得分×0.25 + 满意度得分×0.15

加分：使用 3 种以上策略 +5
扣分：满意度=1 时成交 -10（低满惩罚）

总分范围：理论 0~110
```

| 分数 | 段位 |
|------|------|
| 95-110 | 🥇 谈判大师 |
| 80-94 | 🥈 老练求职者 |
| 60-79 | 🥉 可造之材 |
| 40-59 | 职场小白 |
| 0-39 | 💀 谈崩了 |

## 八、数值平衡分析（v2.0 修正版）

### 可行性验证

**初始状态**：满意度 6、薪资 15K、工时 2、保障 1
**达成 3/4 条件需要**：薪资 ≥ 18K（+3K）、工时 ≥ 7（+5）、保障 ≥ 3（+2）、满意度 > 0

---

### 路径 A：均衡路线（推荐）

```
轮次 1-2：信息搜集 ×2
  → 满意度 -2（剩余4），薪资 +1K（16K），工时 +3（=5），保障 +2（=3）
  ✅ 保障条件已达成

轮次 3-4：反客为主 ×1 + 温和协商 ×1
  → 满意度 -2+1=-1（剩余3），薪资 +2K+0.5K=+2.5K（18.5K）
  → 工时 +1-1=0（=5），保障不变（=3）
  （注意：温和协商的工时-1 抵消了反客为主的工时+1）

轮次 5-6：强硬要求 ×1 + 非谈判内容 ×1
  → 满意度 -3×1.6=-4.8（剩余-1.8→💀 谈崩！）
  ❌ 第 5 轮就翻车了——满意度不够支撑强硬要求
```

**修正后的均衡路线**：

```
轮次 1-2：信息搜集 ×2
  → 满意度 4，薪资 16K，工时 5，保障 3  ✅保障达成

轮次 3：反客为主 ×1
  → 满意度 -2×1.0=-2（剩余2），薪资 +2K（18K） ✅薪资达成
  → 工时 +1（=6）

轮次 4：温和协商 ×1（救火）
  → 满意度 +1（剩余3），薪资 +0.5K（18.5K），工时 -1（=5）

轮次 5：拆穿话术 ×1
  → 满意度 -2×1.6=-3.2（剩余0→💀 谈崩！）
  ❌ 还是翻车——满意度 3 不够支撑第 5 轮的 ×1.6 倍率
```

**最终可行的均衡路线**：

```
轮次 1：信息搜集 → 满意度 5，薪资 15.5K，工时 3.5，保障 2
轮次 2：信息搜集 → 满意度 4，薪资 16K，工时 5，保障 3  ✅保障
轮次 3：温和协商 → 满意度 5，薪资 16.5K，工时 4
轮次 4：反客为主 → 满意度 3，薪资 18.5K，工时 5  ✅薪资
轮次 5：拆穿话术 → 满意度 -2×1.6=-3.2（剩余0→💀）
```

**发现**：即使精心规划，均衡路线在第 5-6 轮极易翻车。这就是平衡的意义——不是随便打都能过。

**真正能活的均衡路线**（需要放弃 1 个条件）：

```
轮次 1：信息搜集 → 满意度 5，薪资 15.5K，工时 3.5，保障 2
轮次 2：信息搜集 → 满意度 4，薪资 16K，工时 5，保障 3  ✅保障
轮次 3：温和协商 → 满意度 5，薪资 16.5K，工时 4
轮次 4：温和协商 → 满意度 6，薪资 17K，工时 3  ⚠️工时倒退了
轮次 5：反客为主 → 满意度 4，薪资 19K，工时 4  ✅薪资
轮次 6：拆穿话术 → 满意度 4-3.2=0.8（≈1），工时 5，保障 4.5
轮次 7：非谈判内容 → 满意度 1+3=4（回血），无进展
轮次 8：强硬要求 → 满意度 4-6=-2→💀
```

**结论**：即使用温和路线苟活，最终也只有 2/4 条件达成（保障+薪资）。工时条件无法在保证存活的前提下达成。**均衡路线能拿 🤝 成交（2/4），但 🏆 高位成交（3/4）仅在菜鸟 HR 面前可行。**

### 路径 B：极限薪资路线（冲 22K）

```
轮次 1：温和协商 → 满意度 7，薪资 15.5K
轮次 2：强硬要求 → 满意度 4，薪资 18K
轮次 3：温和协商 → 满意度 5，薪资 18.5K
轮次 4：强硬要求 → 满意度 2，薪资 21K
轮次 5：非谈判内容 → 满意度 3.5，薪资 21K
轮次 6：强硬要求 → 满意度 3.5-4.8=-1.3→💀
```

**结论**：极限薪资路线必定在第 5-6 轮翻车。22K 需要 3 次强硬要求（-9 满意度），而从初始 6 满意度开始，每张回血牌只回 1-1.5，回血速度远远追不上。

## 九、动态难度调节

### 连击惩罚（防止反复刷同一张牌）

```
连续 2 轮使用相同策略牌 → 本轮效果衰减至 70%
连续 3 轮使用相同策略牌 → 本轮效果衰减至 50%
```

> 注意：惩罚的是效果衰减，而非满意度惩罚减免。连续强硬要求 = 满意度照样扣满，但薪资涨幅打折——双重惩罚。

### 新手保护（前 3 局）

```
首次游玩时，满意度初始值为 7（而非 6）
首次游玩时，温和协商回血量为 +1.5（而非 +1）
```

> 玩家熟悉机制后自动关闭新手保护。

|------|------|
| 🟢 简单 | 初始满意度 7，温和协商回血 +1.5 |

| 🔴 困难 | 初始满意度 5，进攻牌满意度惩罚 ×1.2 |
