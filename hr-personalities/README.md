## 场景相性匹配建议

| 场景 | 推荐性格 | 原因 |
|------|---------|------|
| 初创公司·后端开发 | 笑面虎、老实人、菜鸟 | 创业公司需要亲和力，但也要控制成本 |
| 中型企业·运营岗（待开发） | 刀斧手、笑面虎、机器人 | 运营岗供大于求，HR 议价权强 |
| 大厂·管培生（待开发） | 机器人、笑面虎、刀斧手 | 流程严格，HR 按制度办事 |

> 相性匹配仅为建议，实际可跨场景随机分配以增加 replayability。难度加成系数可用于调整终局评分。

---

## 使用方式

1. 玩家选择场景后，系统从该场景的推荐性格池中随机抽取
2. 将对应性格 MD 注入 LLM System Prompt，与场景配置 MD 合并使用
3. 难度加成系数在终局结算时作用于总分
4. 性格 MD 中的「情绪响应矩阵」用于辅助 LLM 生成符合人设的回复
5. 性格 MD 中的「话术示例」作为 Few-Shot 示例注入
