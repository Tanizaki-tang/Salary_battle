# 场景配置：大厂·管培生（AI Game Master v4.0）

> **场景ID**: scenario_003 | **难度**: ⭐⭐⭐⭐ | **版本**: 1.0
>
> 本文档定义场景的**舞台布景**和**谈判规则**。HR 的性格、语气、话术风格全部由外部性格配置 MD 接管。
>
> 与场景 1、2 的核心差异：大厂校招/管培场景的薪资是"统一批发价"，HR 的角色不是谈判者而是流程执行者。玩家的挑战不在于"说服 HR 给更多"，而在于**在流程中找到可操作空间**和**识别包装在福利里的真实成本**。

## 一、场景客观事实

### 公司与岗位

| 属性 | 值 |
|------|-----|
| 公司 | 星辰集团，已上市互联网平台，10000+人 |
| 岗位 | 管理培训生（产品/运营方向） |
| 城市 | 北京 |
| 招聘背景 | 年度校招管培生计划，全国招聘 200 人，统一薪资标准 |

> 关键差异：这是**校招/管培场景**——薪资弹性极小、福利体系完善但复杂、竞业限制是隐性成本。HR 的底气来自于"200 人的统一标准不可能为你一个人调整"。

### 薪资数据

| 数据 | 数值 | 说明 |
|------|------|------|
| 初始报价 | 15,000/月 | "管培生统一批发价" |
| 市场公允价 | 20,000/月 | 头部大厂管培生市场水平 |
| HR 可批上限 | 22,000/月 | 仅限"特殊人才通道"，极难触发 |
| 法律底线 | 10,000/月 | 仅用于得分计算 |

> 与场景 1/2 的关键区别：虽然薪资空间也是 7K（15K→22K），但 22K 是"特殊人才通道"专属——普通管培生几乎不可能触发。实际可争取的空间约 3K（15K→18K）。

### 福利体系（本场景特有——复杂且包装精美）

| 项目 | 默认状态 | 说明 |
|------|---------|------|
| 五险一金 | 按实际工资全额缴纳 | ✅ 大厂优势：这一点不需要谈判 |
| 公积金比例 | 12%（顶格） | ✅ 大厂优势：这一点也不需要谈判 |
| 住房补贴 | 2,000/月（前 12 个月） | HR 会强调"这相当于变相加薪" |
| 餐补/交通 | 1,500/月（含在总包内） | 已经折算在总包数字里 |
| 培训/轮岗 | 6 个月集中培训 + 2 次轮岗 | HR 会强调"培训价值" |
| 期权/RSU | 无（管培生无股权） | HR 会暗示"转正定岗后有" |
| 竞业限制 | 离职后 12 个月竞业限制 | ⚠️ 隐藏炸弹——玩家很可能不知道这有多大影响 |
| 年假 | 7 天（按国家法定） | 无额外福利假 |
| 试用期 | 6 个月，薪资不打折 | ✅ 大厂优势：试用期薪资不打折 |

> 为什么福利好但博弈空间仍在？因为管培生最大的成本不是薪资高低，而是**竞业限制对后续职业发展的影响**和**轮岗结束后定岗的不确定性**。

## 二、谈判中的博弈要点

> 大厂管培生的博弈逻辑与其他两个场景完全不同。
>
> 场景 1（初创）：HR 有弹性但故意不给 → 博弈是"逼他拿出来"
> 场景 2（中型）：HR 拿体系压价 → 博弈是"绕开体系找空间"
> 场景 3（大厂）：**HR 确实没弹性，但有些东西他不主动告诉你** → 博弈是"发现隐藏信息并利用流程规则"

### 博弈点 1：总包数字的水分

- **博弈本质**：HR 报的"总包 25 万"包含了房补（只发 12 个月）、餐补（已经折算成现金但实际是消费额度）、培训成本（公司花的钱算成你的收益）。真实的、你能自由支配的、长期的现金收入，远低于总包数字。
- **HR 可选策略**：把各种福利折算成现金塞进总包（"月薪 15K + 房补 2K + 餐补 1.5K = 月均 18.5K"）、拿总包跟竞品比（"你看 XX 厂只给 13K 月薪，我们总包远超他们"）、强调隐性福利（"我们的健身房/食堂/培训这些加起来一年也值好几万"）
- **如何判断被攻破**：看玩家是否**将总包拆分为"合同保障的现金收入"和"非现金/临时的福利"**。如果玩家说"房补只发 12 个月，之后断了怎么办"或"请告诉我扣除所有临时性福利后我的保底月薪是多少"——攻破。
- **攻破后的影响**：HR 不能再用膨胀的总包数字来压你。谈判焦点回到长期保底收入。如果基础月薪确实无法调整，HR 可能需要帮你争取"房补延长""签字费"等一次性补偿。

### 博弈点 2：竞业限制——隐形的代价

- **博弈本质**：管培生大概率 1-2 年后跳槽（这是行业的默认剧本）。但公司要求签署 12 个月竞业限制——离职后一年内不能去竞品公司。这意味着如果公司不支付竞业补偿金，你的跳槽成本极高；如果公司支付（通常 30%-50% 月薪），你这一年基本废了。
- **HR 可选策略**：把竞业限制包装成"行业惯例"（"所有大厂都有，不是你一个人"）、放在合同附件里不主动提及、在玩家问到时轻描淡写（"这个是标准条款，大家都签"）
- **如何判断被攻破**：看玩家是否**发现并追问竞业限制条款**。这本身就是一种博弈胜利——大多数新人根本不知道看合同里的竞业条款。如果玩家进一步追问"竞业补偿金标准"和"竞业范围是否合理"——深度攻破。
- **攻破后的影响**：HR 无法再把竞业限制当作"默认条款"来忽略。可能的结果：竞业范围缩小（从全行业改为具体竞品名单）、竞业补偿金标准明确写入合同、或竞业期限缩短至 6 个月。

### 博弈点 3：轮岗结束后的定岗不确定性

- **博弈本质**：管培生的核心承诺是"培养你成为未来的管理者"。但轮岗结束后定到哪个部门、什么级别、薪资是否调整——全部是未知数。HR 会用"看你的表现和公司需求"来回避承诺。
- **HR 可选策略**：画饼（"去年管培生转正后都定 P5 了"）、回避（"这个取决于你的轮岗表现，现在不好说"）、强调灵活性（"你可以自己选方向，这个自由度很多公司没有"）
- **如何判断被攻破**：看玩家是否**要求定岗机制的可量化承诺**。如果玩家问"定岗的考核标准是什么？P5 的薪资范围是什么？如果轮岗表现达到 X 标准，是否保证定岗 P5？"——攻破。如果玩家要求"在 offer 中注明定岗薪资不低于 X 元"——彻底攻破（大厂几乎不可能答应，但你能逼 HR 说出真实情况）。
- **攻破后的影响**：HR 被迫承认定岗存在不确定性。虽然不是每个要求都能满足，但至少你会知道"最坏情况是什么"——这是信息层面的博弈胜利。

### 博弈点 4：签字费和一次性补偿

- **博弈本质**：大厂的月薪确实很难谈——"统一标准"不是借口，是真的。但签字费（sign-on bonus）和搬家费（relocation）是流程外的灵活空间。HR 不会主动提，因为这些预算不在常规 offer 模板里。
- **HR 可选策略**：不提签字费的存在、在被问到时说"我们一般不提供"、把签字费包装成"搬家补贴"（金额低了一个量级）
- **如何判断被攻破**：看玩家是否**明确提到了"签字费"这个概念**。这说明玩家做了功课——知道大厂管培生有不写入标准 offer 的一次性补偿空间。如果玩家能说出竞品给的签字费数字——进一步施加压力。
- **攻破后的影响**：HR 如果在签字费上说"没有"，那就真的没有（大厂不需要在这个问题上撒谎）。但如果公司确实有签字费预算（取决于年份和部门），HR 会说"我帮你问问"。这是本场景中为数不多的纯增量。

### 博弈点 5：管培生不是"员工"而是"培训生"

- **博弈本质**：部分大厂在管培生合同中将其定位为"培训生"而非"正式员工"。这影响——工龄计算（培训期不计入司龄）、离职赔偿条款（培训生离职可能需要赔偿培训费）、转正后的薪资是否按培训期表现重谈。
- **HR 可选策略**：不主动提"培训生"这个身份、在被问到时说"都一样"（实际上不一样）、强调"培训期的价值远超你的想象"
- **如何判断被攻破**：看玩家是否**追问合同中的身份定位和相关条款**。如果玩家问"我的合同身份是正式员工还是培训生？培训期是否计入司龄？如果我在培训期内离职是否需要赔偿培训费？"——攻破。
- **攻破后的影响**：身份问题被摆上桌面后，HR 必须给出明确的合同条款。如果是"培训生"身份，HR 可能需要在培训期计入司龄或免培训赔偿等方面让步。

## 三、HR 让步的硬约束

### 让步幅度上限

| 玩家行为强度 | 单轮最高薪资让步 | 条件 |
|-------------|----------------|------|
| 低（抱怨、轻微不满） | 0 | 大厂不因为情绪让步 |
| 中（有理有据的要求） | 0-500 | 需要有具体理由 |
| 高（攻破博弈点） | 500-1,500 | 攻破总包水分或竞业问题 |
| 签约费/搬家费 | 10,000-30,000（一次性） | 需要攻破签字费博弈点 |

> **大厂场景的让步逻辑不同于其他场景**：月薪让步极小（总空间仅 3K），真正的让步空间在一次性补偿（签字费）和竞业条款优化上。

### 总让步空间管理

- 月薪：15K→18K（实际空间 3K，22K 是理论值但几乎不可触发）
- 签字费：0→30,000（一次性，不在月薪中体现）
- 竞业期限：12个月→6个月（如果可以谈判）
- 福利延长：房补 12个月→18个月（如果可以谈判）

### 非薪资让步优先原则

大厂场景中，**非薪资让步比薪资让步更容易获得**。优先级建议：
1. 竞业限制条款优化（离职后真正影响你的）
2. 签字费（一次性真金白银）
3. 房补延长（持续的真金白银）
4. 月薪调整（最难获得的）

## 七、终局结算

### 得分公式（大厂场景专用）
```
总分 = 薪资得分×0.25 + 博弈得分×0.30 + 福利得分×0.25 + 关系得分×0.20

薪资得分 = (月薪 - 10000) / (22000 - 10000) × 100
博弈得分 = (博弈攻破数 / 5) × 100  （大厂场景博弈质量权重更高）
福利得分 = 签字费得分 + 竞业优化得分 + 房补延续得分
  签字费：每 5,000 = 20 分（上限 60）
  竞业优化：每缩减 2 个月 = 15 分（上限 45）
  房补延续：每延长 3 个月 = 10 分（上限 20）
关系得分 = (满意度 / 100) × 100  （初始 85 已经给了不错的底分）

加分：使用 3 种以上策略 +5
扣分：暴露度>80 则 -10
```

| 得分 | 段位 |
|------|------|
| 95-110 | 🥇 谈判大师 |
| 80-94 | 🥈 老练求职者 |
| 60-79 | 🥉 可造之材 |
| 40-59 | 职场小白 |
| 0-39 | 💀 谈崩了 |

| 薪资空间 | 15K→22K（7K 弹性） | 10K→15K（5K 弹性） | 15K→18K（3K 弹性） |
| HR 策略 | 画饼（期权、成长） | 压价（体系、批量） | 流程（标准、福利包装） |
| 核心博弈 | 期权 vs 现金 | 绩效拆分 + 职级框架 | 总包水分 + 竞业限制 |
| 非薪资焦点 | 试用期、加班费 | 五险一金基数、季度奖 | 竞业条款、签字费、定岗 |
| 薪资谈判弹性 | 高（HR 有预算） | 中（体系内可调） | 低（统一批发价） |
| 非薪资谈判弹性 | 中 | 低（制度僵化） | 高（福利/签字费可调） |
| 学习重点 | 期权 vs 现金、试用期 | 薪资结构拆分、五险一金 | 总包谈判、竞业限制 |
| 难度 | ⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ |

### 为什么大厂场景薪资空间最小但难度最高？

因为大厂的"难"不在数字博弈，而在**信息不对称**——玩家需要识别总包中的水分、发现竞业限制的隐性成本、找到签字费等非标谈判空间。这些不是靠"强硬"能解决的，而是靠"知道问什么问题"。
