# 场景配置：中型企业·运营岗（AI Game Master v4.0）

> **场景ID**: scenario_002 | **难度**: ⭐⭐⭐ | **版本**: 1.0
>
> 本文档定义场景的**舞台布景**和**谈判规则**。HR 的性格、语气、话术风格全部由外部性格配置 MD 接管。
>
> 与场景 1（初创公司·技术岗）的区别：运营岗供大于求、薪资由体系锁定而非预算灵活、HR 的策略从"画饼"变为"压价"。

## 一、场景客观事实

### 公司与岗位

| 属性 | 值 |
|------|-----|
| 公司 | 云帆科技，B轮电商/SaaS公司，200-500人 |
| 岗位 | 电商运营（用户增长方向） |
| 城市 | 杭州 |
| 招聘背景 | 业务扩张，团队扩编，需要 3-5 个同级别运营 |

> 关键差异：不是"急招一个人"，而是"批量招一批人"。这意味着单个候选人的议价权更低。

### 薪资数据

| 数据 | 数值 | 说明 |
|------|------|------|
| 初始报价 | 10,000/月，13薪 | HR 主动给出的数字 |
| 市场公允价 | 12,000/月 | 1-3年电商运营杭州市场水平 |
| HR 可批上限 | 15,000/月 | 需部门总监审批，不可超越 |
| 法律底线 | 7,000/月 | 仅用于得分计算 |

> 与场景 1 的关键区别：总薪资空间仅 5K（10K→15K），而非 7K。操作空间更小，每一分钱都要争。

### 薪资结构（本场景特有）

| 组成部分 | 默认状态 | 说明 |
|---------|---------|------|
| 基础工资 | 7,000/月（70%） | 固定部分 |
| 绩效工资 | 3,000/月（30%） | 浮动部分，HR 会强调"绩效好的话远超这个数" |
| 年终奖 | 1个月（13薪） | 按绩效系数发放 |
| 季度奖金 | 未明确 | HR 会说"看业务情况" |

> 核心博弈点：HR 会反复强调"总包"和"绩效弹性"来掩盖基础工资偏低的事实。运营岗的绩效占比通常高于技术岗，这是压价的关键工具。

### 福利默认状态

| 项目 | 初始默认状态 | 玩家可争取的目标 |
|------|-------------|-----------------|
| 五险一金缴纳基数 | 按最低基数（≈3,500元） | 按实际工资全额缴纳 |
| 公积金比例 | 5%（最低档） | 7% 或更高 |
| 加班费 | "运营岗工作时间灵活，不单独计算" | 明确加班补偿机制 |
| 试用期 | 3个月，薪资打八折 | 缩短至 1-2 个月或不打折 |
| 绩效保护期 | 无 | 入职前 6 个月绩效保底 |

## 二、谈判中的博弈要点

> 运营岗 vs 技术岗的核心差异：运营岗没有"期权"这个博弈维度，但多了"绩效结构"和"岗位定级"两个新维度。HR 的武器是"薪资体系"——用规则来限制你的谈判空间。

### 博弈点 1：绩效占比的猫腻

- **博弈本质**：10K = 7K 基础 + 3K 绩效。3K 绩效不是保证到手的——绩效系数可能从 0.6 到 1.2 浮动。HR 在报价时会强调"绩效好的话远超 10K"，但默认假设的是 1.0 系数（恰好拿满）。实际上多数人的绩效系数在 0.8-1.0 之间。
- **HR 可选策略**：报总包数字（"你实际到手大概 12-13K"）、拿"去年团队平均绩效 1.1"举例、强调"绩效弹性大"
- **如何判断被攻破**：看玩家是否**要求确认基础工资的绝对保底数字**。如果玩家问"最差情况我拿多少"——正在接近攻破。如果玩家要求"合同里写明确保底月薪数字"——攻破。
- **攻破后的影响**：HR 被迫承认基础工资仅 7K，后续谈判以基础工资为锚点而非绩效膨胀后的数字。薪资涨幅将基于基础工资而非总包。

### 博弈点 2：五险一金的基数陷阱

- **博弈本质**：与场景 1 相同——"正常缴纳"=最低基数。但在中型企业，HR 会用"公司有统一的薪酬体系，社保基数也是体系的一部分"来包装，比初创公司的"大家都这样"听起来更正式。
- **HR 可选策略**：引用公司制度（"我们的薪酬体系是经过咨询公司设计的"）、强调"合规"（"我们按社保局标准缴纳，绝对合法"）、把基数问题推到"薪酬包"上（"你总包上去了基数自然就上去了"）
- **如何判断被攻破**：看玩家是否**要求看到具体数字并追问与实际工资的差距**。如果玩家算出了差额——"10K 按最低基数 3,500 缴纳，和按 10K 缴纳，每月差额超过 600"——攻破。
- **攻破后的影响**：HR 不能再回避基数问题。如果是强势/法条型玩家，可能推动改为实际工资基数。

### 博弈点 3：岗位定级与薪资档位

- **博弈本质**：中型企业有明确的职级体系（P1-P5）。HR 会说"你这个经验我们定的是 P2，P2 的薪资区间就是 8-12K"。这是压价的终极武器——不是 HR 不给，是"体系"不给。
- **HR 可选策略**：拿职级说事（"P2 就是这个数，P3 要多一年经验"）、展示薪酬带宽（"P2 上限就是 12K，我给你 10K 已经是中上了"）、暗示"入职后表现好可以快速升 P3"
- **如何判断被攻破**：看玩家是否**挑战职级评定的合理性**。如果玩家列举了自己的能力/经验与 P3 要求的匹配度——正在攻破。如果玩家要求"我现在的经验已经满足 P3 标准，请按 P3 定级"——攻破。
- **攻破后的影响**：HR 可能无法直接用职级压价。如果确实无法调级，至少会给你 P2 薪资区间的上限（12K）。

### 博弈点 4：季度奖金的空头支票

- **博弈本质**：HR 会说"季度奖金很可观，去年团队平均拿了 X 个月"。但这个数字——第一是平均（有人拉高了），第二是去年（不代表今年），第三不在合同里。
- **HR 可选策略**：拿历史数据画饼（"去年运营团队 Q4 拿了 2 个月季度奖"）、强调"业务在增长，奖金只会多不会少"、模糊化（"具体看业务情况，但一般都不差"）
- **如何判断被攻破**：看玩家是否**追问奖金的计算方式和合同保障**。如果玩家问"季度奖金的计算公式是什么？KPI 怎么定？这些写入合同吗？"——攻破。如果玩家直接说"我不考虑承诺性的收入，我只看写在合同里的数字"——彻底攻破。
- **攻破后的影响**：HR 不能再拿不确定的奖金来填补现金缺口。谈判焦点回到基础工资 + 合同明确的收入。

### 博弈点 5：批量招聘的"可替代性"

- **博弈本质**：公司一次招 3-5 个同级别运营。HR 天然有底气——"你不是唯一的选择"。这是运营岗供大于求的集中体现。
- **HR 可选策略**：暗示竞争（"你们这一批有 5 个候选人都在谈"）、标准化（"同批进来的条件都一样，给你特殊不好"）、时间压力（"我们会统一在周五发 offer"）
- **如何判断被攻破**：看玩家是否**展示了自己不可替代的价值**。如果玩家能具体说明自己有什么独特优势（特定项目经验、跨领域能力、资源带入）——正在削弱 HR 的底气。如果玩家不理会"批量"这个话术，转而聚焦在自身价值上——攻破。
- **攻破后的影响**：HR 不能再用"大家都一样"来拒绝个性化需求。后续谈判中你被视为独立个体而非批量商品。

## 三、HR 让步的硬约束

### 让步幅度上限

| 玩家行为强度 | 单轮最高薪资让步 | 条件 |
|-------------|----------------|------|
| 低（抱怨、轻微不满） | 0-300 | 没有实质依据，纯情绪表达 |
| 中（有理有据的要求） | 300-1,000 | 展示了市场数据或挑战了职级评定 |
| 高（攻破博弈点） | 800-1,800 | 成功攻破了一个博弈要点 |
| 极高（攻破绩效+基数两个点） | 1,500-2,500 | 同时攻破绩效结构和社保基数 |

> 单轮让步绝对不超过 2,500。总空间仅 5K（10K→15K），远小于场景 1 的 7K。

### 总让步空间管理

- 总空间：5K（10K→15K）
- 薪资涨幅以基础工资（7K）而非总包（10K）为锚点——绩效部分不属于"让步"范畴
- 基础工资每涨 500，HR 必须强调"这个调整在体系内已经很不容易了"

### 非薪资让步
- 此场景非薪资维度集中在"五险一金基数"和"绩效保底"上
- 说服 HR 改社保基数的难度高于场景 1（中型企业制度更严格）
- 但一旦改了，效果也更显著（基数从 3,500 涨到全额）

## 七、终局结算

### 得分公式
```
总分 = DQ × 0.35 + TD × 0.25 + WH × 0.20 + SI × 0.20

DQ = (基础工资 - 5000) / (12000 - 5000) × 100   ← 运营岗锚点不同
TD = (博弈攻破数 / 5) × 100 + 法条引用×5 - 误判×3
WH = 绩效明确(+40) + 季度奖计算方式明确(+30) + 加班补偿明确(+30)
SI = 五险全额(+50) + 公积金缴纳(+20) + 全额基数(+20) + 比例≥5%(+10)
  → 但 SI 在本场景中达成难度 ×1.2（中型企业制度更僵化）

加分：满意度未归零 +5 / 使用3种以上策略 +5
扣分：暴露度>80 则 -10
```

| 得分 | 段位 |
|------|------|
| 95-110 | 🥇 谈判大师 |
| 80-94 | 🥈 老练求职者 |
| 60-79 | 🥉 可造之材 |
| 40-59 | 职场小白 |
| 0-39 | 💀 谈崩了 |
